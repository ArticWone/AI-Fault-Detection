# Machine Notes

## Connection
- Ethernet port on HMI or inside cabinet
- IP configured via Utility -> Network

## VNC Access
- Use UltraVNC Viewer
- IP: stored locally, do not publish
- Password: stored locally, do not publish

## Known Registers
- 13300: format
- 10142: set format
- 10112: lot number
- 10161: package count

## Goal
Identify fault registers by comparing values during faults.
